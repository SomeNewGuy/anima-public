## Reporting Issues

Open an issue on GitHub with:
- What you expected
- What happened
- Steps to reproduce
- Your environment (OS, Python version, model backend)

## Pull Requests

1. Fork the repo
2. Create a branch (`git checkout -b my-fix`)
3. Make your changes
4. Test locally (`./anima start` and verify)
5. Submit a PR with a clear description

By submitting a pull request, you confirm that you have read and agree to the Contributor License Agreement above.
